# 한국어 콘텐츠 운영 기준

## 페이지명

```text
듄하
```

## 소제목

```text
Dune Awakening Nevermind 길드 도우미
```

## 언어 기준

- 모든 UI 문구는 한국어로 작성합니다.
- 자동 업데이트 데이터도 가능한 한 한국어 표기를 우선합니다.
- `dune.gaming.tools` 등에서 한국어 표시가 가능한 경우 한국어 버전을 우선 반영합니다.
- 수집 요청에는 `Accept-Language: ko-KR`을 적용합니다.
- 장비명/고유명사는 공식 한국어 표기가 확인되지 않으면 원문명을 유지합니다.

## 이유

게임 아이템명은 번역이 흔들릴 경우 검색성과 출처 연결성이 떨어질 수 있습니다.  
따라서 UI/설명/상태/가이드 문구는 한국어로 통일하되, 아이템 고유명사는 확인 가능한 범위에서만 한국어화합니다.
